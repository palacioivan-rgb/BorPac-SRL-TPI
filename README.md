# Bot de Gestión de Vacaciones - BorPac SRL

## Descripción

Este proyecto implementa un chatbot para automatizar el proceso de solicitud de vacaciones en la empresa ficticia BoraPac SRL.

El sistema permite:

- Consultar días disponibles.
- Solicitar vacaciones.
- Validar disponibilidad de días.
- Simular la aprobación del jefe.
- Actualizar el saldo de vacaciones.

## Problema

Actualmente las solicitudes de vacaciones se realizan de forma manual, generando demoras y una alta carga administrativa para Recursos Humanos.

## Solución Propuesta

Se propone la implementación de un chatbot que automatice la validación de solicitudes y las notificaciones correspondientes.
